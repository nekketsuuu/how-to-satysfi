# How to SATySFi

## Prerequisite

* Haskell Stack
* Pandoc >= 2.3
* [SatysfiFilter](https://github.com/nekketsuuu/SatysfiFilter)
  * Run `stack install` to generate `SatysfiFilter-exe`.

## Build

```sh
./build.hs
```
